# tools

Third-party tooling vendored as git submodules.

- `flags-2-env/` — submodule ([flags-2-env](https://github.com/ORESoftware/flags-2-env.git)): the tool that maps CLI flags to environment variables. Used to audit `.cli-flags.toml` in CI and by `scripts/with-flags2env.sh`. Managed via the submodule; do not edit its contents in this repo.
